Customizing Colors

Method 1: Change CSS Variables (Recommended)

Edit the `:root` section in `style.css`:

:root {
--bg-primary: #your-color; /* Main background */
--bg-secondary: #your-color; /* Card backgrounds */
--accent: #your-color; /* Buttons, links, highlights */
--text-primary: #your-color; /* Main text */
--text-secondary: #your-color; /* Secondary text */
}

Typography Customization

Changing Fonts

To use different fonts, update the Google Fonts import and CSS variables:

<!-- In HTML head -->
<link href="https://fonts.googleapis.com/css2?family=YourFont:wght@300;400;600&display=swap" rel="stylesheet">

/* In CSS */
:root {
--font-primary: 'YourFont', sans-serif;
--font-heading: 'YourFont', sans-serif;
}

Animation Adjustments

Modifying Animations

Find animation properties in CSS and adjust:

/* Reduce animation duration */
.element {
transition: all 0.2s ease; /* was 0.3s */
}

/* Disable animations */
* {
animation-duration: 0s !important;
transition-duration: 0s !important;
}

Performance Optimization

Quick Performance Tips

  • Compress all images before uploading
  • Minify CSS and JavaScript files
  • Use WebP format for images when possible
  • Remove unused CSS and JavaScript
  • Enable Gzip compression on your hosting platform
